# 📊 Netflix Data Analysis (EDA)

This project is part of my Data Analytics Internship at CodeAlpha.

## 🔍 Objective
To analyze Netflix dataset and extract meaningful insights using Python.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Key Insights
- More Movies than TV Shows
- Content increased rapidly after 2015
- USA produces most content
- Drama & Comedy are most popular genres
- Most content is rated TV-MA & TV-14

## 📁 Dataset
Netflix Titles Dataset
